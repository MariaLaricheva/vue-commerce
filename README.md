# vue-commerce
## a simple e-commerce app developed with vue

To view app run
```
json-server --watch db.json
```
and in a separate terminal run
```
npm run serve
```


## Project setup
```
npm install
```


### Compiles and minifies for production
```
npm build
```

### Lints and fixes files
```
npm lint
```
